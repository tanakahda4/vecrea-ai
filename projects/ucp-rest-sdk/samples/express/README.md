# UCP REST SDK - Express Sample

Express.js sample that serves the UCP Discovery Profile at `/.well-known/ucp`.

## Getting Started

From the `ucp-rest-sdk` directory:

```bash
cd samples/express
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

Access [http://localhost:3000/.well-known/ucp](http://localhost:3000/.well-known/ucp) to retrieve the UCP Discovery Profile.
