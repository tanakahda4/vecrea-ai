# Projects

Reusable AI-related libraries and projects.

Each subdirectory contains a standalone library or project.

## Structure

- **[wal-sdk/](wal-sdk/README.md)** - Wallet implementation designed for AI Agent operation. Supports CLI, AI Agents (SKILL.md), and Vercel AI SDK. Uses CDP Core for secure signing via email OTP without managing private keys.
- **[wal-sdk-skills/](wal-sdk-skills/README.md)** - Agent Skills for wallet operations (authenticate, send USDC, trade, x402 pay, etc.). Uses wal-sdk/awal CLI.
